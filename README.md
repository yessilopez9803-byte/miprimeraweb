Este proyecto fue desarrollado como parte de un curso de Ciberseguridad con el objetivo de aprender a crear una página web desde cero utilizando HTML, CSS y Bootstrap. A través de este ejercicio se buscó comprender la estructura básica de un sitio web, la función de sus principales elementos, la organización del contenido, la aplicación de estilos para mejorar la experiencia del usuario y la integración de componentes interactivos. Además, permitió adquirir conocimientos fundamentales sobre el funcionamiento de las aplicaciones web, fortaleciendo habilidades que resultan relevantes para el análisis y la comprensión de entornos web dentro del ámbito de la ciberseguridad.
